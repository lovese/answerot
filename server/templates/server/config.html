配置项内容说明
设备序号 通过adb devices获取，如61995125 device，填入61995125即可。
如果adb devices只有一个设备不用配置该项。
屏幕分辨率 width 设备屏幕像素宽度，默认1080px
屏幕分辨率 height 设备屏幕像素高度，默认1920px
百度图片识别API.API_Key 注册百度图片识别应用的API Key
百度图片识别API.Secret_Key 注册百度图片识别应用的Secret Key
